# Sublime Merge 非官方中文语言包

## 归档说明(Archive): 
翻自己的仓库才发现这个五年前的小东西。当时还是太年轻了，不开 issue 讨论就自顾自的上手改东西了，理所当然的 PR 被 Close 了。提交也是很随意的单个不分模块。提交信息也是写的比较神秘。那会还在用 Sublime Text 呢，还是 VSCode 好啊。现在归档这个仓库，就当是留一个学习历史记录吧。来时路这一块。 

> 注意：本语言包并不能汉化所有界面，仅根据官方英文语言包汉化了部分菜单界面。

## 安装方法

### 手动安装

- 打开`Sublime Merge`用户包安装路径：工具栏 `Preference` --> `Browse Packages…`
- 创建名为`Default`的文件夹
- 将本项目打包下载，将全部`*.sublime-menu`文件解压到上述文件夹

### Git 项目克隆

- ```git clone https://github.com/YewFence/sublime-merge-chinese.git <用户包安装路径/Default>```
> 当然也可以直接使用`Sublime Merge`的`Clone Repository`功能
